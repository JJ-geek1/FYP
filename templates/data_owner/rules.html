Cartographie de Protection des Données

Ce panneau récapitule les mécanismes cryptographiques appliqués à chaque attribut avant son stockage ou son traitement par les analystes tiers.

Numéro de CNI / Passeport (`cni`)

Type de donnée : Identifiant Direct (PII)
Pseudonymisation Irréversible Mécanisme : HMAC-SHA256 + Sel

Nom Complet du Citoyen (`nom_patient`)

Type de donnée : Identifiant Direct (PII)
Pseudonymisation Irréversible Mécanisme : HMAC-SHA256 + Sel

Revenu Annuel (`revenu`)

Type de donnée : Attribut Numérique Sensible
Confidentialité Différentielle Mécanisme : Bruit de Laplace ($\Delta S = 50,000$)

Âge (`age`)

Type de donnée : Attribut Numérique
Confidentialité Différentielle Mécanisme : Bruit de Laplace ($\Delta S = 1$)

Note pour l'audit de conformité

Ce système applique nativement les principes du Privacy by Design. Les flux d'ingestion respectent la minimisation des données : aucun intermédiaire réseau ni administrateur de base de données ne peut intercepter les informations nominatives originales.
